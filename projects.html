Movie finder

html, sass, javascript, api | 2022

About the project:

This was a paired programming project and we had to create a web application using JavaScipt and a third-party API. My project partner and I created a movie finder app that generates movie results based on the user's input. We used a second endpoint for our error handling and we embedded an iframe to get the trailer to appear on the page. App is responsive to all screen sizes and is accessible for screen readers.
